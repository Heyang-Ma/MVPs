# Borrowed Time

A puzzle horror game based on the theme “All That Jazz” built in 
Unreal Engine 5.6.1 as part of DES207/DES502 coursework at 
Abertay University, Semester 2 2025/26.

## How to Open the Project    

1. Clone or download this repository (default branch is `audio`)  
2. Open `MVPs.uproject` with Unreal Engine 5.6.1  
3. Press Play
4. If there is Wwise issue, please modify the Wwise version of this project.  

## How to Play  

- WASD: Move  
- Mouse: Look  
- Left Mouse: Fire taser  
- E: Interact  
- C: Crouch  
- Q: Switch shoulder
- R: Switch backward sight
- ESC: Pause menu
